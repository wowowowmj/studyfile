    자료명, 항목명 입력

Google Sheets에서 데이터 수정하기

구조: 웹페이지 ↔ Supabase (실시간) ↔ Google Sheets (수동 새로고침)
  • 1
    스프레드시트 열기 → 상단 메뉴 데이터 관리 → 최신 데이터 불러오기 클릭
  • 2
    상태편집 탭에서 흰색 열만 수정 (저장 버튼 없음, 자동 저장)
  • 3
    웹페이지에서 확인하려면 브라우저 새로고침
⚠️ 회색 열(학년·자료명·항목 등)은 수정하지 마세요. 새로고침 시 덮어써집니다.

Apps Script 트리거 확인 (자동저장이 안 될 때)

  • 1
    스프레드시트 → 확장 프로그램 → Apps Script
  • 2
    왼쪽 시계 아이콘 클릭
  • 3
    onEdit 트리거가 있는지 확인. 없으면 + 트리거 추가 → 함수: onEdit / 이벤트: 수정 시 → 저장
