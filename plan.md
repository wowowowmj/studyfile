## 3. 주요 기능 목록

### 3-1. 상단 고정 내비게이션
- `position: sticky` 로 스크롤 시 상단 고정.
- 탭: **학년별 / 자료유형별 / 담당자별** (전체현황·로드맵·우선작업 탭은 제거됨).
- 클릭 시 해당 섹션으로 부드러운 스크롤 이동 (`scroll-behavior: smooth`).

### 3-2. 전체 완료율 바
- 헤더에 전체 항목 대비 완료 수를 계산해 진행률 바로 표시.
- 체크 상태 변경 시 실시간 갱신.

### 3-3. 자료준비 로드맵
- 5개 시간 구간으로 자료를 분류.
- 구간별 완료율에 따라 초록/노랑/빨강 색상 구분.
- 각 구간 hover 시 포함 자료 목록과 완료 상태를 툴팁으로 표시.

### 3-4. 우선작업 체크리스트 패널
- 로드맵 기준 미완료 항목을 자동 수집해 목록화.
- 남은 작업 수를 배지로 표시.
- 이 패널에서 체크하면 메인 대시보드와 실시간 연동.
- 담당자가 있는 경우 태그로 표시.

### 3-5. 학년별 진행률 카드
- 3·4·5·6학년 각각 색상 구분 (옐로우→오렌지→브라운 계열).
- 학년 단위 미니 진행 바와 퍼센트 표시.
- **학년 섹션 기본 접힘** — 화살표 버튼 클릭 시 펼치기/접기.
- 학기 > 카드(자료유형) 계층 구조로 구성.

### 3-6. 단원별 체크리스트 카드
- 카드 유형: 달크업 세트(study) / 단원평가(unit) / 핵심콕VOD(vod).
- 원형 SVG 진행률 그래프 (r=36, `stroke-dashoffset` 방식).
- 진행률 색상: 빨강(~33%) → 주황(34~66%) → 오렌지(67~99%) → 초록(100%).
- 100% 완료 시 "모든 준비가 완료되었습니다!" 배지 표시.

### 3-7. 담당자 지정
- 각 체크리스트 항목 옆 드롭다운으로 담당자 선택.
- 선택지: 민주 / 찬희 / 대표님 / **미정(기본값)**.
- 미정이 아닌 담당자 선택 시 `has-assignee` 클래스 → 앰버 밑줄 강조.
- 변경 시 `CustomEvent('assignee-updated')` 발행 → 담당자별 섹션 자동 갱신.
- `localStorage` 키: `assignee-{cardId}-{idx}`.

### 3-8. 항목 상세 모달
- 각 항목의 `···` 버튼 클릭 시 모달 오픈.
- 편집 가능 필드: 담당자, 마감일, 메모, 제출링크.
- 저장 시 `localStorage`에 반영 + DOM 즉시 업데이트.
- `window.openDetailModal(cardId, idx)` 전역 노출.

### 3-9. 자료유형별 완료율 섹션
- 달크업 세트 / 단원평가 / 핵심콕VOD 3가지 유형별 완료율 집계.
- 프로그레스 바와 완료 건수/전체 건수 표시.

### 3-10. 담당자별 담당업무 섹션
- 민주 / 찬희 / 대표님 / 미정 4개 패널.
- **전체 기본 접힘** — 화살표 클릭으로 개별 펼치기/접기.
- 접힘 상태는 `collapsedCards` Set에 보관 → DOM 재렌더링 후에도 유지.
- 완료된 항목은 취소선으로 표시.
- **모바일**: 2열 그리드로 재배치, 화살표 버튼이 이름 아래 별도 행에 표시.

### 3-11. 데이터 버전 관리
- `DATA_VERSION = 'v4-vod-added'`.
- 버전 변경 시 기존 `localStorage` 상태 자동 초기화.

## 4. localStorage 키 구조

| 키 | 값 | 설명 |
|----|-----|------|
| `card-{id}` | JSON 배열 (boolean[]) | 카드별 체크 상태 |
| `assignee-{cardId}-{idx}` | 문자열 (이름) | 항목별 담당자 |
| `detail-{cardId}-{idx}-deadline` | 문자열 | 항목 마감일 |
| `detail-{cardId}-{idx}-memo` | 문자열 | 항목 메모 |
| `detail-{cardId}-{idx}-link` | 문자열 | 항목 제출 링크 |
| `studyDash_version` | 문자열 | 데이터 버전 |
